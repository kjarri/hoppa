# Hoppa

A 2D platformer game built with JavaScript and HTML5 Canvas.

## Play Online

**https://kjarri.github.io/hoppa/**

## Controls

- **A / D** - Move left/right
- **W / Space** - Jump
- **Arrow Keys** - Shoot

## Development

To run locally:

```bash
# Open index.html in a browser
# Or serve with a local server
npx serve .
```
